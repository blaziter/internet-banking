# Internetové bankovnictví

### Zpracovali: Petr Tran, Pavel Tomšík

- aplikace požaduje
  - meson - build systém
  - kompilační nástroje z MINGW2

- build a spuštění aplikace

```bash
# v projektové složce
meson build
meson compile -C build
```
